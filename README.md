# Kepler Exoplanet Search Results

## Descripción del Proyecto
Este proyecto forma parte del **Diplomado en Ciencia de Datos**. 
Su propósito es analizar el conjunto de datos acumulativo de la misión Kepler de la NASA, con el fin de:

- Caracterizar y predecir la disposición de los "Objetos de Interés de Kepler" (KOI).
- Explorar relaciones entre variables astronómicas relevantes.
- Implementar técnicas de limpieza, imputación y modelado predictivo.
- Evaluar patrones que puedan ayudar a identificar exoplanetas candidatos.

Los KOI representan señales detectadas por Kepler que podrían corresponder a exoplanetas, por lo que su estudio es fundamental en astrofísica moderna.

##  Dataset
Los datos provienen del **Kepler Space Observatory**.
- **Fuente:** [NASA Exoplanet Archive / Kaggle](https://www.kaggle.com/datasets/nasa/kepler-exoplanet-search-results)
- **Registros:** ~9,564 objetos.
- **Variables clave:** Periodo orbital, radio del planeta, temperatura de equilibrio, dimensiones de la estrella anfitriona.
